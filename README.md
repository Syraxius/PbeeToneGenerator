# Pbee Tone Generator

This tone generator is used for replicating the experiment featured in this YouTube video by Steve Mould:

https://www.youtube.com/watch?v=C-V1uXeyGmg

## Usage:

1. Clone the repository
2. Double-click on `src/index.html`
3. Enjoy!
